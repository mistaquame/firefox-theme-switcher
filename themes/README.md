# Firefox Theme Switcher - Themes

Drop each downloaded Firefox theme into its own folder here.

Each folder should contain:
- `userChrome.css` — browser UI styling
- `userContent.css` — new tab / home page colors (optional)

Example:
```
themes/
  FoxOne/
    userChrome.css
    userContent.css
  Safari-like/
    userChrome.css
    userContent.css
```

The app scans these folders and shows them as clickable cards.
